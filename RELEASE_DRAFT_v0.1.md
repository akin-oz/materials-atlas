# GitHub Release Draft — v0.1.0

## Materials Atlas v0.1.0

Materials Atlas is an open knowledge system for Computational Materials Science. It is designed for software engineers with a materials background who want a disciplined route into computational materials research, scientific software, materials informatics, and AI for materials.

### Highlights

- A high-level roadmap for navigating the field.
- A 16-module, competency-based curriculum from mathematical recovery to a capstone research project.
- Canonical references for thermodynamics, phase diagrams, and statistical mechanics.
- A reusable Mermaid diagram library with stable identifiers and editorial ownership.
- Curated indexes for books, papers, software, videos, researchers, and institutions.
- Editorial, licensing, copyright, and citation foundations suitable for an open scientific project.

### Repository Philosophy

Materials Atlas is curated rather than exhaustive. Pages, diagrams, and resources must earn their place by reducing uncertainty for a reader. The repository favors primary sources, stable concepts, explicit relationships, and maintainable structure over accumulation.

### Getting Started

1. Read [README.md](https://github.com/akin-oz/materials-atlas#readme) for the repository scope and navigation.
2. Use [ROADMAP.md](https://github.com/akin-oz/materials-atlas/blob/main/ROADMAP.md) for the field-level progression.
3. Start the detailed [Software Engineer → Computational Materials Scientist curriculum](https://github.com/akin-oz/materials-atlas/tree/main/roadmaps/software-engineer-to-computational-materials-scientist).
4. Follow [STUDY-METHOD.md](https://github.com/akin-oz/materials-atlas/blob/main/roadmaps/software-engineer-to-computational-materials-scientist/STUDY-METHOD.md) while working through modules.

### Roadmap

v0.1.0 establishes the project's editorial and navigational foundation. Future work will focus on carefully curated domain pages and stronger links between modules, references, and the broader research ecosystem. A Zenodo archive and DOI will be added after this release is preserved.
