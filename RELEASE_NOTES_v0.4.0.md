# Materials Atlas v0.4.0 — Interactive Foundations

## Overview

v0.4.0 introduces the official Jupyter notebook series as the first interactive learning layer of Materials Atlas. The notebooks make foundational concepts executable, visual, and reproducible while preserving the Atlas's documented learning path.

## Highlights

- `notebooks/` is now a first-class repository layer with a defined naming convention, progression, execution policy, and quality standard.
- The notebook environment declares JupyterLab, NumPy, and Matplotlib dependencies.
- Every included notebook is designed to run from a clean kernel and combines explanation, focused code, visualization, experimentation, reflection, and a Materials Atlas connection.

## Educational Philosophy

Notebooks complement rather than replace the Markdown knowledge base. Markdown roadmaps and references remain canonical: they own stable explanations, curriculum sequence, and curated context. Notebooks provide interactive exploration of a focused concept, allowing learners to vary assumptions, run calculations, inspect results, and record interpretations.

## Included Notebooks

- `00-python-environment.ipynb` — Jupyter workflow, notebook state, NumPy, Matplotlib, and reproducible execution habits.
- `01-linear-algebra.ipynb` — vectors, transformations, linear systems, eigenvectors, and their computational materials applications.
- `02-atomic-bonding.ipynb` — energy landscapes, interatomic interactions, bonding types, and simplified interatomic potentials.
- `03-crystal-structures.ipynb` — lattices, bases, unit cells, packing, density, directions, planes, and anisotropy.

## What's Next

The interactive curriculum will continue with Thermodynamics, Statistical Mechanics, Scientific Python, and Numerical Methods. No release dates are promised.
