# References

> Canonical concepts, diagrams, and durable explanations used across Materials Atlas.

## Purpose

The `references/` directory holds stable material that supports multiple roadmaps, domains, and resource indexes.

Reference pages explain a concept once and give later pages a canonical destination. They are not learning modules, resource lists, or substitutes for primary sources.

## Navigation

- [thermodynamics/](thermodynamics/) contains canonical references for thermodynamic quantities, phase diagrams, and recurring terminology.
- [diagrams/](diagrams/) contains the authoritative Mermaid diagram library and its editorial rules.

## Editorial Boundary

Add a reference page only when the same explanation, notation, or diagram would otherwise be repeated across the Atlas.

Use a roadmap to teach a sequence, a domain page to map a field, and a resource index to curate external work. Use `references/` for durable internal explanations that those sections can cite.

See [../EDITORIAL.md](../EDITORIAL.md) for the repository-wide inclusion standard.
